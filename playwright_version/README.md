# Playwright версия - Оптимизированная

## Структура:

- **`payment_step1.py`** - Шаг 1: Сумма + Способ + Продолжить
- **`payment_step2.py`** - Шаг 2: Заполнение данных (будет дальше)
- **`payment_step3.py`** - Шаг 3: Капча + QR (будет дальше)

## Запуск:

```bash
cd playwright_version
python3 payment_step1.py
```

## Преимущества Playwright:

1. **Быстрый ввод** - прямо в React state без посимвольного ввода
2. **Авто-ожидания** - не нужны `time.sleep()`
3. **Умная навигация** - автоматически ждет загрузки
4. **Стабильнее** - меньше race conditions

## Ожидаемое время:

- **Шаг 1**: 2-3 секунды (вместо 5-7s на Selenium)
- **Полный процесс**: 10-12 секунд (вместо 37s)
