# Статус бота @AxisPay_bot

## ✅ Текущее состояние

Бот работает с кнопками "Платные подписки" и "Баланс", но:
- **Пополнение баланса ЗАБЛОКИРОВАНО** - выводится сообщение об ошибке
- **Подписка на каналы ЗАБЛОКИРОВАНА** - выводится сообщение об ошибке
- **MulenPay полностью отключен** - нет интеграции с платежной системой

## 📋 Главное меню

Кнопки в главном меню:
1. 📄 Политика конфиденциальности ✅
2. 📋 Пользовательское соглашение ✅
3. 🔮 Гороскоп AI ✅ (бесплатно)
4. 💬 Поддержка ✅
5. 💎 Платные подписки ⚠️ (показывает ошибку)
6. 💰 Баланс ⚠️ (показывает баланс, но пополнение заблокировано)

## ⚠️ Заблокированные функции

При нажатии на:
- **"Пополнить баланс"** → "⚠️ Пополнение временно недоступно"
- **Любой канал (Крипто/Фитнес/Питание/AI)** → "⚠️ Подписка временно недоступна"

Сообщение об ошибке:
```
⚠️ Пополнение временно недоступно

К сожалению, функция пополнения баланса временно отключена.
Пожалуйста, свяжитесь с поддержкой для получения дополнительной информации.
```
